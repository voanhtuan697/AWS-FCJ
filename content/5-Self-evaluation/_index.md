During my internship at AMAZON WEB SERVICES VIETNAM COMPANY LIMITED from 09/03/2026 to 02/05/2026, I had the opportunity to work directly on a real-world cloud project titled "Personal Expense Manager (PEM)" — a full-stack web application built entirely on AWS.
Throughout this internship, I applied the knowledge learned in school — software engineering, cloud computing, and system design — to build a complete personal finance management platform using Next.js, NestJS, and over 15 AWS services.

I contributed to the development of key modules in the system, including:

- Designing and implementing the full-stack architecture with Next.js 14 (Frontend) and NestJS (Backend) following modern design patterns

- Building core features such as transaction management, budget tracking, financial account management, dashboard analytics, and report generation

- Integrating AWS services including Cognito, RDS PostgreSQL, ElastiCache Redis, S3, SES, Lambda, EventBridge, ECS Fargate, ALB, ECR, Amplify, CloudFront, VPC, and CloudWatch

- Implementing serverless budget alert system using AWS Lambda + EventBridge + SES for automated email notifications

- Setting up CI/CD pipeline with AWS CodePipeline + CodeBuild for automated testing and deployment

- Supporting the preparation of architecture diagrams, system documentation, and AWS cost estimation

Beyond technical skills, I developed communication, teamwork, documentation, and problem-solving abilities by working closely with supervisors and adapting to professional workflows.
