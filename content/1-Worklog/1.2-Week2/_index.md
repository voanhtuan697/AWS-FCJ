### Week 2 Achievements

- Apply IAM principles: creating users, roles, and access policies.
- Know how to selectively assign permissions and organize permission groups according to needs.
- Understand the relationship between users, groups, roles, and policies in AWS.
- Successfully deploy, connect, and operate EC2 instances in the Free Tier.
- Perform operations such as starting, stopping, and deleting instances via the EC2 Console.
- Master VPC knowledge: Subnet, Route Table, Internet Gateway, NAT Gateway.
- Understand routing mechanisms and how to securely partition virtual networks.
- Learn about AWS Site-to-Site VPN: how to connect on-premises to VPCs via tunnels.
- Experience Amazon Bedrock, become familiar with Claude and Titan; understand how to write effective prompts.
- Strengthen Python programming skills alongside AWS service practice.
- Gain an overview of Compute (EC2, Lambda), Networking (VPC, VPN), and AI/ML (Bedrock).
