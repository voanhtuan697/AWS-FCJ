### Week 4 Goals

- Learn about and optimize storage on **Amazon S3**.
- Enhance skills in managing, securing, and optimizing S3 performance.

### Week 4 Achievements

- Understand **Amazon S3**: storage classes, costs, performance, and durability for each class.
- Practice advanced **S3** management: Cross-Region Replication, Lifecycle Policy, Versioning.
- Strengthen S3 security: IAM, Bucket Policy, SSE/KMS encryption, Block Public Access.
- Set up **CloudFront** and **Global Accelerator**, test latency and user experience.
- Learn about **Storage Gateway (hybrid)**, **FSx for Windows**, and **FSx for Lustre** for high-performance workloads.
- Understand **Integration & Messaging** mechanisms: SQS, SNS, Step Functions on AWS.
