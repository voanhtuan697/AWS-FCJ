### Week 5 Goals

- Deploy and manage containers on AWS with ECS, EKS, and Docker.
- Understand the concepts and benefits of serverless architecture.
- Design a serverless application integrating Lambda, API Gateway, S3, and DynamoDB.
- Learn about AWS database services: RDS, Aurora, DynamoDB, Redshift, and ElastiCache.

**Time Period:** 06/04/2026 - 10/04/2026

### Week 5 Achievements

- Deploy containers using **ECS (Fargate)** and **EKS**; manage images with ECR.
- Build event-driven **AWS Lambda** functions; integrate with API Gateway to create RESTful endpoints.
- Design a complete serverless application: Lambda + DynamoDB + S3 + Step Functions.
- Work with RDS (relational), DynamoDB (NoSQL), Aurora, Redshift (analytics), and ElastiCache.
- Combine containers, serverless systems, and databases in the same deployment environment using a DevOps mindset.
