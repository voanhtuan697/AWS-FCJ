Nhật ký công việc

Tuần 1 (09/03 - 13/03): Làm quen với AWS và các dịch vụ cơ bản

  • Tiếp cận nền tảng AWS Cloud và các nhóm dịch vụ chủ chốt (Compute, Storage, Networking, Database)
  • Thực hành trên AWS Free Tier, Console và CLI
  • Xây dựng kỹ năng thao tác cơ bản với Amazon EC2

Tuần 2 (16/03 - 20/03): AWS IAM, triển khai EC2, mạng VPC và các khái niệm cơ bản

  • Vận dụng AWS IAM để kiểm soát danh tính và quyền truy cập
  • Triển khai và vận hành EC2 Instance thực tế
  • Khám phá Amazon VPC và Site-to-Site VPN, củng cố nền tảng lập trình C#

Tuần 3 (23/03 - 27/03): High Availability, Scalability, RDS, Aurora, ElastiCache, Route 53

  • Nắm bắt High Availability và Scalability trong môi trường cloud
  • Làm việc với các dịch vụ database được quản lý: RDS, Aurora, ElastiCache
  • Cấu hình Route 53 cho DNS và điều hướng lưu lượng; nghiên cứu các mô hình kiến trúc AWS điển hình

Tuần 4 (30/03 - 03/04): Lưu trữ Amazon S3, CloudFront và các dịch vụ tích hợp

  • Nghiên cứu Amazon S3, CloudFront, Global Accelerator, Storage Gateway và FSx
  • Tích hợp hệ thống qua SQS, SNS và Step Functions
  • Thực hành thiết kế giải pháp lưu trữ và phân phối nội dung

Tuần 5 (06/04 - 10/04): Containers trên AWS, kiến trúc Serverless và Database

  • Triển khai và quản lý container với ECS, EKS và Docker
  • Xây dựng ứng dụng Serverless kết hợp Lambda, API Gateway, S3 và DynamoDB
  • Mở rộng kiến thức về RDS, Aurora, Redshift và ElastiCache

Tuần 6 (13/04 - 16/04): Data & Analytics, Machine Learning và Monitoring

  • Tìm hiểu Data & Analytics với Athena, Glue và Kinesis
  • Giám sát hệ thống bằng CloudWatch, CloudTrail và nâng cao kỹ năng quản lý IAM

Tuần 7 (20/04 - 24/04): Bảo mật AWS, VPC và Disaster Recovery cho ứng dụng quản lý chi tiêu

  • Áp dụng bảo mật dữ liệu tài chính trên AWS (KMS, IAM, Cognito)
  • Cấu hình và tối ưu Amazon VPC (subnet, route table, NAT Gateway)
  • Thiết lập Security Groups và Network ACLs kiểm soát truy cập
  • Sử dụng AWS WAF & Shield bảo vệ ứng dụng
  • Triển khai AWS Backup cho RDS PostgreSQL
  • Cấu hình Multi-AZ và thử nghiệm failover
  • Xây dựng kế hoạch Disaster Recovery (Cross-Region)

Tuần 8 (27/04 - 01/05): Kiến trúc hệ thống nâng cao và tối ưu AWS cho ứng dụng quản lý chi tiêu

  • Thiết kế kiến trúc hệ thống production-ready cho PEM
  • Tích hợp S3 (upload hóa đơn, lưu báo cáo)
  • Tích hợp Redis (ElastiCache) để cache dữ liệu dashboard
  • Tối ưu database và luồng xử lý nghiệp vụ (transactions, budgets)
  • Áp dụng AWS Well-Architected Framework
  • Tham khảo kiến trúc mẫu (3-tier, financial system)
  • Hoàn thiện Proposal (Architecture, Implementation Plan, Cost)
